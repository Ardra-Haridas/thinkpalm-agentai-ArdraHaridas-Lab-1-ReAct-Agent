\#ReAct Agent using Gemini API (Google Colab)



This project implements a simple \*\*ReAct (Reasoning + Acting) Agent\*\* using the Gemini API.



The agent can:

\- Perform calculations

\- Look up definitions

\- Answer general questions



\---



\## Features



\- Uses Gemini API for reasoning

\- Implements ReAct framework (Thought → Action → Answer)

\- Supports tool usage:

&#x20; - Calculator

&#x20; - Dictionary

\- Runs interactively in Google Colab
